## 📞 学习建议

1. **一定要动手** — 命令行是练出来的
2. 不要死记命令，理解用途更重要
3. 善用 `--help` 和 `man` 查看帮助
4. 用 AI 帮你解释命令输出

## 📋 核心命令速查

| 类别 | 命令 |
|------|------|
| 文件操作 | `ls`, `cd`, `cp`, `mv`, `rm`, `mkdir` |
| 文本查看 | `cat`, `less`, `head`, `tail` |
| 文本处理 | `grep`, `cut`, `sort`, `uniq`, `wc` |
| 管道 | `\|` |
| 重定向 | `>`, `>>`, `<` |
| 权限 | `chmod`, `chown` |
| 进程 | `ps`, `top`, `kill` |
