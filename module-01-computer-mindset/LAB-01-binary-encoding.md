## 🔬 实验步骤

### 第一部分：二进制计数（15 分钟）

#### 步骤 1：理解二进制

十进制（我们日常用的）：
```
位值：  1000  100  10   1
        10³   10²  10¹  10⁰

数字：   3    4    5    6
        = 3×1000 + 4×100 + 5×10 + 6×1
        = 3456
```

二进制（计算机用的）：
```
位值：  8   4   2   1
        2³  2²  2¹  2⁰

数字：  1   0   1   1
        = 1×8 + 0×4 + 1×2 + 1×1
        = 8 + 0 + 2 + 1
        = 11（十进制）
```

#### 步骤 2：练习转换

**任务：** 将以下十进制数转换成二进制

| 十进制 | 二进制（你来填） |
|--------|-----------------|
| 5      | ? |
| 10     | ? |
| 15     | ? |
| 32     | ? |

<details>
<summary>点击查看答案</summary>

- 5 = 0101 (4+1)
- 10 = 1010 (8+2)
- 15 = 1111 (8+4+2+1)
- 32 = 100000 (32)

</details>

#### 步骤 3：用 Python 验证

```bash
# 打开 Python
python3
```

```python
# 十进制转二进制
print(bin(5))    # 输出：0b101
print(bin(10))   # 输出：0b1010
print(bin(15))   # 输出：0b1111

# 二进制转十进制
print(int('101', 2))    # 输出：5
print(int('1010', 2))   # 输出：10
```

`0b` 前缀表示这是二进制数。

### 第二部分：文本编码（15 分钟）

#### 步骤 1：查看字符的编码

```bash
python3
```

```python
# 查看字符的 Unicode 编码
print(ord('A'))    # 输出：65
print(ord('B'))    # 输出：66
print(ord('0'))    # 输出：48

# 中文字符
print(ord('你'))   # 输出：20320
print(ord('好'))   # 输出：22909

# 数字转字符
print(chr(65))     # 输出：A
print(chr(20320))  # 输出：你
```

#### 步骤 2：编码一句话

**任务：** 把你的名字转换成 Unicode 编码

```python
name = "你的名字"
for char in name:
    print(f"{char}: {ord(char)}")
```

示例输出：
```
张：24352
三：19977
```

#### 步骤 3：理解 UTF-8 字节

```python
text = "Hello"

# 编码成 UTF-8 字节
encoded = text.encode('utf-8')
print(encoded)          # 输出：b'Hello'
print(list(encoded))    # 输出：[72, 101, 108, 108, 111]

# 每个字符的编码
for char in text:
    utf8_bytes = char.encode('utf-8')
    print(f"{char}: {list(utf8_bytes)}")
```

输出：
```
H: [72]
e: [101]
l: [108]
l: [108]
o: [111]
```

**观察：** 英文字符在 UTF-8 中用 1 个字节表示。

#### 步骤 4：中文字符的字节

```python
text = "你好"

encoded = text.encode('utf-8')
print(list(encoded))
# 输出：[228, 189, 160, 229, 165, 189]
# 每个汉字用 3 个字节！
```

#### 练习 3：文件大小估算

**问题：** 一篇 1000 个汉字的文章，用 UTF-8 编码，大约多大？

**计算：**
```
1 个汉字 ≈ 3 字节（UTF-8）
1000 汉字 × 3 字节 = 3000 字节
3000 字节 ÷ 1024 ≈ 2.93 KB（约 3 KB）
```

> 💡 注意：1 KB = 1024 字节（二进制标准），不是 1000 字节。硬盘厂商通常用 1000 换算，但操作系统用 1024 换算。

**验证：**
```bash
# 创建一个测试文件
echo "这是一个测试文件，用于验证中文编码的大小。" > test.txt

# 查看文件大小
ls -l test.txt

# 查看字节数
wc -c test.txt
```
