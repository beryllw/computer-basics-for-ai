# 模块 1：计算机思维基础

> 建立"计算机如何工作"的心智模型

---

## 🎯 模块目标

学完本模块后，你将能够：

- [ ] 理解什么是"计算"
- [ ] 理解数据如何用二进制表示
- [ ] 理解抽象层次的概念
- [ ] 用算法思维思考问题

---

## ⏱️ 预计完成时间

- **正常**: 3-5 天（每天 1 小时）

## 📞 学习建议
