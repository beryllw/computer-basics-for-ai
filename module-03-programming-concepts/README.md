# 模块 3：编程概念与代码阅读

> 理解程序结构，能读懂 AI 生成的代码

---

## 🎯 模块目标

学完本模块后，你将能够：

- [ ] 理解编程的核心概念（变量、函数、控制流）
- [ ] 读懂 Python 代码的基本结构
- [ ] 理解 AI 生成的代码并能修改
- [ ] 编写简单的 Python 脚本
