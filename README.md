# 计算机基础入门课程

> 为零基础学习者设计的计算机基础课程，帮助你为学习大模型和 AI Agent 做好准备

---

## 🎯 课程目标

本课程面向**零基础或弱基础**的学习者，目标是：

1. **建立正确的计算机心智模型** — 理解计算机如何工作
2. **掌握实用的基础技能** — 命令行、代码阅读、数据处理
3. **理解 AI 系统基础** — 为后续学习大模型和 Agent 做准备
4. **学会与 AI 协作** — 把 AI 当作强大的学习伙伴

**学完本课程后，你将能够：**

- ✅ 理解计算机的核心概念（计算、数据、抽象、算法）
- ✅ 熟练使用 20+ 个 Linux 命令
- ✅ 读懂 AI 生成的 Python 代码
- ✅ 理解 API、网络、数据格式等基础概念
- ✅ 理解 AI 模型、Token、Prompt 等核心概念
- ✅ 设计有效的人机协作工作流

## 📖 如何学习

### 推荐学习流程

1. **阅读概念文档** — 理解核心思想，不要死记硬背
2. **完成实验 (LAB)** — 动手实践，验证理解
3. **做练习题** — 巩固知识，发现盲点
4. **通过 Checkpoint** — 检查是否掌握本模块内容
5. **继续下一模块** — 知识是累积的，确保理解后再前进

### 和 AI 协作学习

本课程设计中，AI 是你的**学习伙伴**：

```
✅ 应该用 AI：
- 解释不理解的概念（让 AI 用不同类比）
- 检查实验结果是否正确
- 生成额外练习题
- 解答具体问题
- 审查你的代码

❌ 不应该用 AI：
- 直接代替你思考
- 直接给实验答案
- 跳过基础概念学习
```

### 学习时间建议

| 学习节奏 | 每天时间 | 完成课程 |
|----------|----------|----------|
| 快速     | 2-3 小时  | 6 周     |
| 正常     | 1-2 小时  | 8-10 周  |
| 轻松     | 30-60 分钟 | 12-15 周 |
