## 🤖 路径 A：AI / 大模型方向

适合对 AI、大模型、Agent 开发感兴趣的学习者

### 第一阶段：AI 基础（2-4 周）

**学习内容**:
- Python 编程基础（如果还不够熟练）
- 机器学习基础概念
- 神经网络入门

**推荐资源**:
- [ ] 吴恩达《机器学习》课程（Coursera）
- [ ] fast.ai 实用深度学习
- [ ] 3Blue1Brown 神经网络系列（B 站/YouTube）

**实践项目**:
- 用 Python 实现简单的神经网络
- 使用预训练模型做图像分类

---

### 第二阶段：大模型应用（4-8 周）

**学习内容**:
- Transformer 架构基础
- Prompt Engineering
- API 调用和集成
- RAG（检索增强生成）基础

**推荐资源**:
- [ ] 李宏毅《生成式 AI》课程（YouTube）
- [ ] DeepLearning.AI 短课程
- [ ] LangChain 官方文档和教程

**实践项目**:
- 构建一个基于 API 的 AI 应用
- 实现一个简单的问答系统
- 用 LangChain 创建 AI Agent

---

### 第三阶段：Agent 开发（8-12 周）

**学习内容**:
- Agent 架构设计
- 工具调用（Function Calling）
- 多 Agent 协作
- 记忆和规划

**实践项目**:
- 创建一个能自主完成任务的 Agent
- 构建多 Agent 协作系统
- 开发一个实用的 AI 工作流

**进阶方向**:
- 大模型微调（Fine-tuning）
- 模型部署和优化
- 垂直领域应用

## 🎯 学习建议

### 1. 持续学习策略

```
✅ 建立学习习惯
- 每天固定时间学习
- 小步快走，不要贪多
- 记录学习日志

✅ 项目驱动学习
- 学以致用
- 做有挑战但能完成的项目
- 把项目放进作品集

✅ 加入社区
- 参与讨论
- 向他人学习
- 分享你的知识
```

### 2. 避免常见陷阱

```
❌ 教程地狱：不停看教程，从不动手
✅ 解决：每学一个概念，立刻动手实践

❌ 完美主义：一定要完全理解才继续
✅ 解决：接受模糊理解，在实践中深化

❌ 孤立学习：只靠自己，不问问题
✅ 解决：加入社区，寻求帮助

❌ 盲目跟风：什么火学什么
✅ 解决：明确目标，专注一条路径
```

### 3. 建立作品集

```
无论选择哪个方向，都要：

1. GitHub 账号
   - 存放你的代码
   - 记录学习历程

2. 技术博客（可选）
   - 记录学习心得
   - 加深理解

3. 个人项目
   - 展示你的能力
   - 面试时的谈资
```
