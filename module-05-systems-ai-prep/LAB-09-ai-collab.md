# 实验 LAB-09：AI 协作练习

> 通过与 AI 协作完成学习任务

---

## 🎯 实验目标

- [ ] 能够用 AI 解释不懂的概念
- [ ] 能够用 AI 生成练习题
- [ ] 能够用 AI 审查代码
- [ ] 能够用 AI 改进工作流

---

## 📋 准备工作

本实验需要访问 AI 助手（如本课程中的 AI）。

---

## 第一部分：用 AI 学习概念 (20 分钟)

### 任务 1.1：解释概念

**你的任务**：用 AI 解释一个你不完全理解的概念。

**示例 Prompt**：
```
我正在学习计算机基础，对 [概念名] 还不太理解。
我已经知道 [相关知识]。
请用生活中的类比帮我解释 [概念名]。
```

**记录**：
- 你选择的概念：__________
- AI 使用的类比：__________
- 你现在是否理解了？__________

### 任务 1.2：多方式解释

**你的任务**：让 AI 用不同方式解释同一概念。

**示例 Prompt**：
```
谢谢，我用类比理解了。现在能给我：
1. 正式的定义
2. 一个代码示例
3. 一个实际应用场景
```

**记录**：
- 正式定义：__________
- 代码示例要点：__________

---

## 第二部分：用 AI 生成练习 (25 分钟)

### 任务 2.1：生成练习题

**你的任务**：让 AI 生成针对某个主题的练习题。

**示例 Prompt**：
```
我刚学习了 Python 的变量和数据类型。
请生成 5 道练习题帮我巩固：
- 2 道选择题
- 2 道代码阅读题
- 1 道编程题

请先出题，我回答后再给答案和解析。
```

**完成练习**：
- 记录你的答案
- 对照 AI 给的答案
- 统计正确率

### 任务 2.2：针对性练习

**你的任务**：针对薄弱环节请求更多练习。

**示例 Prompt**：
```
我在 [某类型] 题目上错了。
能再给我 3 道类似的题目吗？
这次请附带详细解析。
```

---

## 第三部分：用 AI 审查代码 (25 分钟)

### 任务 3.1：代码审查

**你的任务**：让 AI 审查你写的代码。

**示例 Prompt**：
```
请审查我写的这个 Python 函数：

```python
def calculate_average(numbers):
    total = 0
    for n in numbers:
        total += n
    return total / len(numbers)
```

请检查：
1. 逻辑是否正确
2. 有没有边界情况没考虑
3. 代码风格是否规范
4. 有没有更简洁的写法
```

**记录 AI 的建议**：
- __________
- __________

### 任务 3.2：根据反馈修改

**你的任务**：根据 AI 的建议修改代码。

**修改后代码**：
```python
# 你的修改版本
```

**验证**：
```bash
# 测试你的代码
python3 your_code.py
```

---

## 第四部分：设计学习工作流 (30 分钟)

### 任务 4.1：咨询 AI

**你的任务**：让 AI 帮你设计学习某主题的工作流。

**示例 Prompt**：
```
我想学习 [主题，如：API 调用]。
我目前的水平是 [描述你的水平]。
我每周可以投入 [时间]。

请帮我设计一个学习工作流，包括：
1. 学习步骤
2. 推荐资源
3. 实践项目建议
4. 如何检验学习效果
```

### 任务 4.2：实施并反馈

**你的任务**：按照 AI 设计的工作流学习，然后给 AI 反馈。

**示例 Prompt**：
```
我按照你设计的工作流学习了 [时间]。
我的感受是：
- 有用的部分：...
- 困难的部分：...
- 收获：...

请根据我的反馈调整学习计划。
```
